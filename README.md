# Internship-work-1


 🐼 Pandas

Core Python library for data manipulation & analysis

Works with DataFrames & Series — think of it as Excel but in code

Handles loading, cleaning, filtering, grouping & transforming data effortlessly

Foundation of almost every ML & data science pipeline



🔢 NumPy

Core Python library for numerical computing

Works with arrays & matrices — faster and more efficient than plain Python lists

Powers math operations: linear algebra, statistics, random sampling & reshaping

The backbone behind Pandas, Scikit-learn, TensorFlow and most ML libraries
